# Klondike Finance Interface

This is front-end repository of the [klondike.finance](https://klondike.finance/).

### Getting started

1. Add to `env.local`

```
REACT_APP_INFURA_KEY=...
REACT_APP_NETWORK=mainnet
```

2. Install modules

```
yarn
```

3. Start the app

```
yarn start
```
